# Gene-disease-network-analysis
This project analyzes shared genetic mechanisms among Obesity, Type 2 Diabetes Mellitus, and Hypertension using network science and graph-based machine learning. Centrality analysis and Node2Vec-based clustering reveal latent genetic similarities across metabolic disorders.
